# Computational Fractals and Information Theory Algorithms

This repository contains implementations of fractal geometry, dynamical systems, statistical physics simulations, and data compression algorithms developed as part of academic research and computational mathematics coursework.

The project combines theoretical concepts from nonlinear dynamics, information theory, statistical physics, and fractal geometry with practical implementations in Python.


---

# Implemented Algorithms and Models

## Fractal Geometry

- Hilbert Curve (space-filling curve)
- Koch Curve variations
- Sierpinski Triangle using L-systems
- Stochastic L-system vegetation simulation
- Fractal landscape generation (Diamond-Square algorithm)
- Barnsley Fern using Iterated Function Systems (IFS)

These algorithms demonstrate recursive geometry, self-similarity, and procedural generation.

---

## Dynamical Systems

- Newton Method implementation
- Complex Newton fractals and basins of attraction
- Diffusion Limited Aggregation (DLA) dendritic growth simulation

These simulations illustrate nonlinear dynamics, convergence behavior, and fractal attractors.

---

## Statistical Physics Simulations

- Percolation model simulation
- Experimental estimation of the critical exponent Î² = 5/36
- Forest fire propagation model and visualization

These models reproduce phase transitions and critical phenomena described in statistical physics.

---

## Data Compression Algorithms

- Complete implementation of LZ77 compression algorithm
- Fractal image compression based on Fisher (1995)
- Language-specific text compression algorithm for Vietnamese

Includes compression, decompression, and performance evaluation.

---

## Information Theory

- Fifth approximation text generator based on Abramson (1963)
- Probabilistic language modeling using real corpus data

Demonstrates entropy-based modeling and stochastic language generation.

---

## Research Project

Includes a theoretical and computational study of the Hofstadter Butterfly fractal spectrum.

This project explores:

- Fractal structure in quantum systems
- Self-similarity in spectral distributions
- Computational visualization of mathematical physics phenomena
